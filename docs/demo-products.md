## 设计原则

1. **不使用真实品牌商标**：所有商品使用通用描述，避免侵权
2. **不写虚假功效承诺**：尤其是健康、美妆、母婴类商品
3. **价格合理**：参考市场真实价格范围
4. **卖点真实**：基于产品实际功能描述
5. **风险提示**：特殊类别（美妆、母婴、食品）有明确的风险提示

## 使用方式

### API 调用
```bash
# 获取所有 Demo 商品列表
GET /api/demo-products

# 获取指定 Demo 商品
GET /api/demo-products/0

# 获取指定类别的 Demo 商品
GET /api/demo-products?category=数码科技
```

### 前端一键填充
前端页面提供 30 个 Demo 商品的快捷按钮，点击即可自动填充商品信息并触发视频生成。
