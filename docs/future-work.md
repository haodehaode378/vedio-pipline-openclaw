# ShopMotion v1 - 未来规划

## 短期（1-2 个月）

### 1. 接入真实 LLM
- 替换 MockLLMProvider 为 OpenAI/Claude API
- 实现更智能的卖点提取和分镜生成
- 支持多语言文案生成

### 2. 接入真实 TTS
- 替换 MockTTSProvider 为 MiniMax/Azure TTS
- 实现语音合成与时间轴同步
- 支持多种语音风格

### 3. 真实视频渲染
- 集成 FFmpeg 进行视频编码
- 支持商品图片/视频素材上传
- 实现真实视频导出

### 4. 数据持久化
- 引入 SQLite 或 PostgreSQL
- 任务历史持久化
- 用户配置保存

## 中期（3-6 个月）

### 5. 模板系统增强
- 扩展到 20+ 模板家族
- 支持自定义模板编辑
- 模板 A/B 测试

### 6. 素材管理
- 商品图片上传与管理
- 品牌素材库
- 字体库管理

### 7. 批量生成
- 支持批量商品导入
- 队列管理系统
- 并行渲染

### 8. 用户系统
- 用户注册登录
- 项目管理
- 团队协作

## 长期（6-12 个月）

### 9. AI 增强
- 智能商品图片生成
- 自动配乐推荐
- 智能剪辑节奏

### 10. 平台化
- 多平台分发（抖音、快手、小红书）
- 数据分析与效果追踪
- ROI 分析

### 11. 开放平台
- 模板市场
- API 开放
- 插件系统
