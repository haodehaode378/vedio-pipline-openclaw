# ShopMotion v1.1 - 前端设计文档

## 概述

ShopMotion 前端基于 React + TypeScript 构建，提供完整的商品输入、模板推荐、分镜预览和字幕导出功能。

## 页面结构

### 1. 顶部导航栏
- 项目标题和 Logo
- 模板系统统计（家族数、配置数、组合数）
- 未验证功能提示

### 2. 标签页导航
- **🚀 创建视频**：商品输入和生成
- **📜 任务历史**：查看历史任务
- **🎨 模板系统**：浏览模板家族

### 3. 创建视频页面

#### 左侧：商品输入区域
- **商品名称**：文本输入
- **商品类别**：下拉选择（20个类别）
- **价格/原价**：数字输入
- **商品描述**：多行文本
- **目标人群**：文本输入
- **核心卖点**：多行文本（每行一个）
- **营销目标**：下拉选择（6种目标）
- **视频风格**：下拉选择（15种风格）
- **视频时长**：下拉选择（15s/30s/45s）

#### 左侧：Demo 商品一键填充
- 30 个 Demo 商品的快捷按钮
- 点击自动填充并触发生成

#### 右侧：结果展示
- **任务状态卡片**：进度条和状态文字
- **模板推荐卡片**：推荐家族、置信度、理由
- **分镜脚本展示**：
  - 商品信息摘要
  - 卖点列表
  - 字幕时间轴
  - 场景列表（含场景类型、标题、旁白、字幕）
  - SRT 导出链接

### 4. 任务历史页面
- 任务列表表格
- 商品名称、类别、模板、状态、时间
- 点击跳转查看详情

### 5. 模板系统页面
- 统计卡片（家族数、配置数、组合数）
- 模板家族卡片网格
- 每个卡片显示：名称、描述、适用类别、密度、动效

## 组件设计

### ProductForm（商品表单）
- 响应式布局
- 表单验证
- 类别联动风格推荐
- 一键清空/重置

### DemoProductsBar（Demo 商品栏）
- 横向滚动展示
- 悬停高亮
- 加载状态禁用

### JobStatusCard（任务状态卡片）
- 实时进度条
- 状态文字映射
- 错误信息展示

### TemplateRecommendationCard（模板推荐卡片）
- 家族名称和置信度
- 推荐理由
- 支持的场景类型标签
- 回退提示

### StoryboardDisplay（分镜展示）
- 商品信息摘要
- 卖点卡片
- 字幕时间轴表格
- 场景列表（含类型图标）
- SRT 导出按钮

### TemplateSystemView（模板系统视图）
- 统计卡片
- 家族卡片网格
- 类别标签展示
