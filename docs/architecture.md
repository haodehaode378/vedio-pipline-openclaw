# ShopMotion v1 - 系统架构文档

## 1. 项目概述

ShopMotion 是一个基于 React + TypeScript + Node.js + Remotion 的电商带货短视频自动生成系统。v1 版本在 MVP 基础上重点增强了 Remotion 模板系统，实现了 15 个模板家族、45+ 模板配置、可扩展到 100+ 模板组合的架构。

## 3. 模块边界

### 3.1 商品模块 (`src/modules/product/`)
- 职责：商品输入类型定义、商品类别配置
- 依赖：无
- 被依赖：生成模块、模板系统、前端

### 3.2 模板系统 (`src/modules/templates/`)
- 职责：模板家族定义、模板配置、模板注册、模板解析
- 子模块：
  - `families/` - 15 个模板家族定义
  - `configs/` - 45 个模板配置（每家族 3 个变体）
  - `registry/` - 模板注册表
  - `resolver/` - 模板解析器
  - `presets/` - 商品类别配置预设

### 3.3 生成模块 (`src/modules/generation/`)
- 职责：Mock LLM 生成卖点和分镜脚本
- 依赖：商品模块、模板系统
- 输出：Storyboard JSON

### 3.4 任务系统 (`src/modules/jobs/`)
- 职责：任务创建、状态管理、任务查询
- 实现：InMemoryJobStore（内存存储）

### 3.5 字幕模块 (`src/modules/captions/`)
- 职责：字幕时间轴生成、SRT 导出、JSON 导出

### 3.6 旁白模块 (`src/modules/voiceover/`)
- 职责：Mock TTS、旁白时间轴生成

### 3.7 通知模块 (`src/modules/notification/`)
- 职责：飞书通知、Mock 通知
- 安全：只读环境变量，不硬编码

### 3.8 Remotion 渲染层 (`src/modules/remotion/`)
- 职责：视频 Composition、场景渲染、共享组件
- 子模块：
  - `shared/` - 共享视觉组件
  - `templates/` - 场景渲染器
  - `compositions/` - Composition 入口

### 3.9 Demo 数据 (`src/modules/demo-data/`)
- 职责：12 个 demo 商品、示例分镜

### 3.10 前端 (`src/modules/frontend/`)
- 职责：用户界面、商品表单、模板展示、任务管理

### 3.11 服务端 (`src/modules/server/`)
- 职责：Express API 服务、路由、任务处理流程

## 4. 数据流

```
用户输入商品信息
    ↓
POST /api/generate
    ↓
创建任务 (InMemoryJobStore)
    ↓
MockLLM 生成卖点 → SellingPoint[]
    ↓
TemplateResolver 推荐模板 → TemplateRecommendation
    ↓
MockLLM 生成分镜 → Storyboard (含模板信息)
    ↓
任务状态更新为 completed
    ↓
前端展示：模板推荐 + 分镜脚本 + 字幕时间轴
    ↓
可选：导出 SRT 字幕 / 查看旁白时间轴
```

## 5. 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| 前端 | React 18 + TypeScript | 用户界面 |
| 构建 | Vite 5 | 前端构建与开发服务器 |
| 后端 | Express 4 + TypeScript | API 服务 |
| 视频 | Remotion 4 | 视频模板渲染 |
| 类型 | TypeScript strict | 类型安全 |
| 运行 | tsx | TypeScript 直接运行 |
