# ShopMotion v1 - 演示流程

## 演示准备

1. 启动后端: `npm run dev:server`
2. 启动前端: `npm run dev:frontend`
3. 打开浏览器: http://localhost:5173

## 演示流程

### 1. 展示模板系统（2分钟）
- 点击「🎨 模板系统」标签
- 展示 15 个模板家族卡片
- 展示模板统计数据（15 家族、45 配置、135+ 组合）
- 说明三层架构：家族 → 变体 → 配置

### 2. Demo 商品一键填充（1分钟）
- 回到「🚀 创建视频」标签
- 展示 12 个 Demo 商品按钮
- 点击「蓝牙降噪耳机 · ¥599」一键填充

### 3. 任务生成过程（1分钟）
- 观察任务状态从 queued → generating_script → resolving_template → completed
- 展示进度条动画

### 4. 模板推荐展示（1分钟）
- 展示模板推荐卡片
- 说明推荐理由、置信度、支持的场景类型

### 5. 分镜脚本展示（2分钟）
- 展示完整分镜脚本
- 说明 6 个场景：Hook → Product Reveal → 卖点1 → 卖点2 → 价格 → CTA
- 展示字幕时间轴
- 展示 SRT 导出功能

### 6. 任务历史（1分钟）
- 切换到「📜 任务历史」标签
- 展示已生成的任务列表
- 点击查看历史任务详情

### 7. Remotion 预览（可选，1分钟）
- 启动 Remotion Studio: `npm run dev:remotion`
- 展示视频预览效果

## 不同类别商品演示

为展示模板系统的适配能力，可依次演示：
1. **数码科技** → TechClean 科技简洁风
2. **美妆护肤** → SoftBeauty 美妆柔和风
3. **食品饮料** → FreshFood 食品清新风
4. **健身运动** → SportEnergy 运动活力风
5. **母婴用品** → BabyCare 母婴安心风

每个类别展示不同的模板推荐和视觉风格。
